Como Executar:

    1- Pré-requisitos:

        Certifique-se de ter o Python3 instalado e as bibliotecas:
        - numpy
        - pandas
        - scikit-learn
        - matplotlib
        - seaborn


    2- Executando os experimentos:

        Com as dependências instaladas, basta rodar o script principal que executará os testes do KNN, do K-Means, fará as comparações, imprimirá os resultados e salvará os gráficos na pasta 'images/' com:

            python3 main.py

        Caso não funcione, tente  apenas:

            python main.py
